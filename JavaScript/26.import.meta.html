import.meta

import.meta是一个给JavaScript模块暴露特定上下文的元数据属性的对象。有ECMAScript实现，原型是null，并且默认带有一个url对象，这个对象可扩展，它的属性都是可写，可配置，可枚举的

<script type="module" src="./index.js"></script> // 必须是module

console.log(import.meta); // {url: 'http://127.0.0.1:5500/index.js'}
