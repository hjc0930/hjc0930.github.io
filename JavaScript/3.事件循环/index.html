事件循环机制

1.JavaScript执行机制

  • 同步：在执行到同步代码时，会按照顺序执行直接输出
  • 异步：在执行到异步代码时，会将异步代码的回调函数存入事件队列中，等所有同步代码执行完成后再执行异步代码

2.宏任务&&微任务

异步任务分为宏任务和微任务，微任务执行时机先于宏任务
