work-break用于设置单词在行内的断行行为
```css
/* 关键字值 */
word-break: normal;
word-break: break-all; /* 允许在单词内换行 */
word-break: keep-all;  /* 只能在半角空格或连字符处换行 */

/* 全局值 */
word-break: inherit;
word-break: initial;
word-break: unset;
```
