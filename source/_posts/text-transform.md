**text-transform**属性指定如何将元素的文本设置为大写，可以全部大写或全部小写，也可以单独对每一个单词进行操作

| 属性 | 含义 |
| --- | --- |
| capitalize | 首个字符大写 |
| uppercase | 所有字符大写 |
| lowercase | 所有字符小写 |
| none | 禁止转换 |
| full-width | 按照一般东亚文字(如中文或日文)对齐 |
